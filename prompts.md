# Prompt Documentation

## Objective

Generate professional website copy for a local coffee shop.

## AI Tool Used

ChatGPT

## Prompt

(You are a professional website copywriter.

Create complete website copy for a modern local coffee shop called Brew Haven Café.

Requirements:

• Friendly and premium tone
• Target audience:
  - Students
  - Families
  - Professionals
• Mention freshly brewed coffee, pastries, free Wi-Fi, cozy atmosphere.
• Include:

1. Hero Section
2. About Section
3. Why Choose Us
4. Services
5. Menu Highlights
6. Customer Testimonials
7. Contact Section
8. Strong Call To Action

Keep the language persuasive, modern and website-ready.)

## Improvements

- Requested premium tone.
- Added customer-focused messaging.
- Included clear call-to-action.
- Structured output into website sections.

## Outcome

Generated homepage content, service descriptions, and CTA suitable for a modern café website.
